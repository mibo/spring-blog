= Playground for Spring (Reactive) and Gradle

== Build and run application

  * Build via `gradle bootJar`
  * Run afterwards via `java -jar build/libs/blog-0.0.1-SNAPSHOT.jar`
  * Open browser at link:http://localhost:8080[]
